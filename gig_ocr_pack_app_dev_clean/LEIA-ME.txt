GIG OCR PACK (para gig_app)
===============================

Objetivo:
- permitir OCR no gig_app sem instalacao com privilegio de administrador.

Conteudo:
- Tesseract-OCR
- qpdf-portable
- ghostscript
- ocr-runtime\python (runtime Python com pacotes OCR locais)
- activate_ocr_pack.bat
- run_gig_with_ocr_pack.bat
- run_gig_app_dev_with_ocr_pack.bat
- check_ocr_pack.bat

Uso recomendado no computador do trabalho:
1. Copie esta pasta inteira para qualquer local de usuario.
2. No repo do gig_app do trabalho, mantenha o codigo atualizado.
3. Rode check_ocr_pack.bat para validar o pack.
4. Rode run_gig_with_ocr_pack.bat <CAMINHO_DO_REPO_DO_APP>.

Alternativa:
- rode activate_ocr_pack.bat para abrir um terminal com as variaveis de OCR ativas;
  depois execute o gig_app manualmente no mesmo terminal.
- o pack usa tmp local em GIG_OCR_TEMP_ROOT para evitar depender do TEMP do sistema.
